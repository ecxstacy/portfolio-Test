This is a Next.js starter for Patrick Nyutu Mbugua. It includes pages, simple components, styles, and a Netlify CMS admin stub (for Netlify deploy + Identity + Git Gateway).

## Quick start (local)
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Open http://localhost:3000

## Deploy
Connect this repository to Vercel or Netlify. For Netlify CMS, follow the included Netlify CMS instructions in the root README or admin/README.md
